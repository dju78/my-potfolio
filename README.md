# Professional Portfolio - Daramola Joseph Omoyele

## Overview
This is a modern, responsive professional portfolio website showcasing the comprehensive CV and professional experience of Daramola Joseph Omoyele - Economist, Senior Data Analyst, and Regulatory & Policy Analytics Specialist.

## Features

### Design
- **Premium Modern Aesthetics**: Vibrant gradients, smooth animations, and glassmorphism effects
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Intersection Observer API for scroll-triggered animations
- **Interactive Elements**: Hover effects, parallax scrolling, and dynamic transitions

### Sections
1. **Hero Section**: Eye-catching introduction with animated gradient orbs and key statistics
2. **About**: Executive value proposition and core capabilities
3. **Experience**: Interactive timeline showcasing 15+ years of professional experience
4. **Research**: Academic work including MSc and undergraduate dissertations
5. **Skills**: Comprehensive capability stack across 4 key areas
6. **Publications**: Book publication and media commentary
7. **Contact**: Multiple contact methods with availability status

## Technology Stack
- **HTML5**: Semantic markup with SEO best practices
- **CSS3**: Modern CSS with custom properties, gradients, and animations
- **Vanilla JavaScript**: No frameworks, pure JS for optimal performance
- **Google Fonts**: Inter (body) and Playfair Display (headings)
